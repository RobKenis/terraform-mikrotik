# OpenTofu - Mikrotik

This repository contains configuration for the Mikrotik devices I use at home. This might be a little overengineered.

Credits to <https://github.com/mirceanton/mikrotik-terraform> for the inspiration and some of the code.

## IP Ranges

- Management: 10.0.0.1/24
- DHCP Networks (192.168.96.0/20):
  - Guest: 192.168.96.0/24
  - Trusted: 192.168.97.0/24
