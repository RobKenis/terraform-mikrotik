# CSS326-24G-2S+RM

## System

Configure static IP to `10.0.0.2` as SwOS does not support DHCP client in a VLAN. Enable `Independenent VLAN Lookup`

## VLANs

| VLAN ID | Name       | Ports |
| ------- | ---------- | ----- |
| 1000    | Management | 2, 8  |
| 1100    | Trusted    | 2, 7  |
| 1200    | Guest      | 2, 6  |
